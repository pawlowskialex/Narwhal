# Narwhal

*A bulletproof, buttery-smooth Reddit client*

### Features

  - Top/Best/New browsing
  - Nested subreddit navigation (tap on subreddit name)
  - Thumbnail preview (tap on thumbnail)
  - Post link preview (tap on post)

### Runnning

  - Open Xcode project
  - Change signing identity
  - Run

You know the drill.

### Dependencies

 - None
